# WindowKeys

Plugin for fetching window handle using keyboard shortcuts. Overlay is displayed on all visible windows with key combination for selection.

```
+---------------+--------------+-------------------------------+
|               |              |                               |
|               |              |                               |
|     [A]       |     [R]      |    [S]   +-----------+        |
|               |              |          |    [T]    |        |
|               |              |          +-----------+        |
|               |              |                               |
+---------------+--------------+-------------------------------+
```


## Usage

Type the displayed letter(s) to select a window or **Escape** to cancel.
